## SQL 与 MongoDB 术语对比

| 术语/概念 |     SQL     |                   MongoDB |
| --------- | :---------: | ------------------------: |
| 数据库    |  Database   |                  Database |
| 表        |    Table    |                Collection |
| 行        |     Row     | Document 或 BSON Document |
| 列        |   Column    |                     Field |
| 索引      |    Index    |                     Index |
| 表关联    | Table Joins | \$lookup 或内嵌 Documents |
| 主键      | Primary Key |   Primary Key, 默认为\_id |
| 聚合运算  | Aggregation |      Aggregation Pipeline |
