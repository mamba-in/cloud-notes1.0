# bash 常用命令

## 文件操作

```bash
ls      列出文件
ls -l   列出文件详情
ls -a   列出所有文件,包含隐藏文件
touch xx    创建xx文件
cat filename    列出指定文件的内容
mv f1 f2    将文件/文件夹f1重命名为f2
mv f1 f2    将文件/文件夹f1重命名为f2
```

## 文件夹操作

```bash
mkdir xx    创建xx文件夹
mkdir -p firstname/secondname    创建多层目录
rmdir dirname    删除名为dirname的空目录,有内容不能删
rm  -rf  dirname    强制删除目标目录
```
